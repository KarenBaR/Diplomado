Causa raíz del 34% con diferencia
Causa principal — NVE con tope de Venta Manifestada
96 casos · 100% del DIFF

El Excel calcula la Venta Neta Evaluada como:

NVE = max(VPA × FI, Venta Manifestada)
CEM = NVE × Margen − GF − GFam

El API solo usa VPA × FI, sin aplicar el tope. En 96 casos la Venta Manifestada es mayor, generando una diferencia promedio de S/ 23,188 en NVE.

Corrección: agregar Venta Manifestada como input y aplicar el max(). Esto llevaría la precisión del 65.7% al ~100%.

Conclusiones y próximos pasos
01

El API calcula correctamente el 65.7% de los casos

184 de 280 con diferencia ≤ 2%. La mediana de diferencia es exactamente 0.00%, lo que confirma que la fórmula base es correcta.

02

La causa principal es la Venta Manifestada como tope (96 casos)

El Excel aplica NVE = max(VPA × FI, VM). Agregar la Venta Manifestada como input en el API y aplicar ese tope llevaría la precisión al ~100%.

03

El GF siempre debe declararlo el ejecutivo

Usar cuota_sf de la BD reduce la precisión de 66% a 13.9%. El ejecutivo incluye deudas en otras entidades que la BD no ve. Diferencia media: S/ 3,478.

04

Cuota SF de la BD es confiable (100%)

cuota_sf = cuota_total_rrll + cuota_total_empresa exactamente en los 439 registros. Cero error en este campo.

05

Tabla FI actualizada difiere en 16 casos (5.7%)

Tramos S/30K–200K cambiaron significativamente respecto a la tabla original del Excel. Revisar si el ajuste es intencional o debe revertirse para esos rangos.
